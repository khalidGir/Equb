# UI Color Palette (Approved)

## Status Colors

- PRIMARY: #2E7D32
- SUCCESS: #388E3C
- WARNING: #F57C00
- ERROR: #D32F2F
- NEUTRAL: #F5F5F5

## UI Colors

- BACKGROUND: #FFFFFF
- SURFACE: #FAFAFA
- TEXT PRIMARY: #212121
- TEXT SECONDARY: #757575
- BORDERS: #E0E0E0
- HIGHLIGHT: #E8F5E9

## Accent Colors

- ACTIVE: #4CAF50
- INACTIVE: #BDBDBD

## Usage Rules

1. Max 3 colors per screen
2. Minimum contrast 4.5:1
3. Use icons with color for accessibility
4. Green = good, Red = problem, Orange = attention
