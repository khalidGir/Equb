# Design Principles (Approved)

## Accessibility

- Minimum 16pt text
- 44x44px touch targets
- WCAG AA contrast
- All icons have labels
- Full screen-reader support

## Low-Tech Optimization

- Minimal images
- <1MB per active session
- No heavy animations
- Clear offline mode
- Skeleton loading screens

## Cultural Considerations

- Amharic first
- RTL supported
- Culturally aligned colors
- Ethiopian numerals optional
- Ethiopian calendar optional

## Trust Indicators

- Full transparency of actions
- Shared visibility between admin and members
- Confirm irreversible actions
- Easy data export
- Accessible help
